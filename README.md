# diverdanger
A canvas game about a diver in serious danger

# Development Requirements

- Nodejs

# Install

```ssh
$ npm install
```

# Running

```ssh
$ npm run dev
$ npm run prod
```

## Move assets

```ssh
$ npm run assets:dev
$ npm run assets:prod
```

For dev, check localhost:8181 in your browser.

For production execute on local or server the dist/prod folder content
